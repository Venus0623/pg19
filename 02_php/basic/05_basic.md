## no41.php

### devices.txt

次のファイルを作成してください。

```
Central Processing Unit
Hard Disk Drive
Solid State Drive
```

次の変数があります。

```php
<?php
$file = "devices.txt";
```

devices.txtファイルを読み込んで、実行結果のとおり出力してください。

### 実行結果

```

Hard Disk Drive
Solid State Drive
```
