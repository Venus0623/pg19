# エクササイズ - 関数 - 配列関数

## ar_ex1.php

次の配列データがあります。

```php
<?php
$cards = ["J", "Q", "K", "A"];
???
print_r($cards);
```

次の実行結果となるようにPHPプログラムを作成してください。

### 実行結果

```
> php ar_ex1.php
Array
(
    [0] => J
    [1] => Q
    [2] => K
)
```

> 先頭から3つの要素を取得します。
