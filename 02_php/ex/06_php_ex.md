## array_ex5.php

次の配列があります。

```php
$names = ["Alice", "Bob"];
```

次の実行結果となるようにPHPプログラムを作成してください。

### 実行結果

```
> php array_ex5.php
array(2) {
  [0]=>
  string(3) "Bob"
  [1]=>
  string(5) "Alice"
}
```

> "Bob"と"Alice"と入れ替えます。

### ヒント

```php
<?php
$names = ["Alice", "Bob"];

// TODO

var_dump($names);
```
